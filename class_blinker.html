Detailed Description

Klasa odpowiedzialna za miganie/stan kierunkowskazów.

Klasa powiazania z QML przez Q_PROPERTY Sygnaly:

-bool stateChanged - mowi o zalaczeniu/wylaczeniu kierunkowskazu -double blinkChanged - uzywane z timerem, mruganie kierunkowskazu gdy jest zalaczony
